Rubrik CDM REST API to RSC GraphQL API Mapping

The following table lists the mapping between Rubrik CDM REST API endpoints and their corresponding GraphQL query and mutation alternatives.

The values in the CDM 9.0 Availability column specify whether the REST API endpoints will continue to be available in Rubrik CDM versions 9.0 and later. Following are the possible values:

  • Available, indicating that you can continue to use the REST API endpoint in Rubrik CDM versions 9.0 and later.

  • Restricted, indicating that you will need to switch over to the GraphQL endpoint corresponding to the REST API endpoint.

For certain REST endpoints the value for the corresponding GraphQL endpoint is N/A. N/A indicates that a GraphQL API query or mutation is currently not available for the specific REST endpoint.

Finding GraphQL operations in the reference schema

To locate a GraphQL query or mutation in the GraphQL reference schema, complete the following steps.

  1. Access the Rubrik GraphQL query reference schema, or the Rubrik GraphQL mutation reference schema.

  2. Click inside the GraphQL Schema definition panel.

  3. Use Ctrl-F (Windows and Linux) or Cmd+F (Mac OS) to launch the browser search box for the HTML page.

  4. Enter the GraphQL operation name.

The search results will include similar matches in the left navigation pane. Ignore these until you find a match in the GraphQL Schema definition panel.
