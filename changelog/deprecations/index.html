Deperecations

2023/06/01

  • Enum value GROUP_NAME was deprecated in enum TargetQueryFilterField.

    Reason: We do not use archival groups currently. Scheduled for removal after 2024/04/21.

  • Field ownerOrgName was deprecated in ClusterSlaDomain type.

    Reason: This field has been deprecated in favor of ownerOrg field. Scheduled for removal after 2024/05/11.

  • Field awsComputeSettings was deprecated in Query type.

    Reason: This query is no longer supported. Scheduled for removal after 2024/04/29.

  • Enum value IS_MANAGED_BY_AUTO_AG was deprecated in enum TargetQueryFilterField.

    Reason: We do not use archival groups currently. Scheduled for removal after 2024/04/21.

  • Field updateAwsComputeSetting was deprecated in Mutation type.

    Reason: This mutation is no longer supported. Scheduled for removal after 2024/04/29.

  • Field createAwsComputeSetting was deprecated in Mutation type.

    Reason: This mutation is no longer supported. Scheduled for removal after 2024/04/29.

  • Field userNotifications was deprecated in Query type.

    Reason: This query is deprecated. Scheduled for removal after 2024/04/27.

  • Field deleteAwsComputeSetting was deprecated in Mutation type.

    Reason: This mutation is no longer supported. Scheduled for removal after 2024/04/29.

  • Enum value IS_ARCHIVED was deprecated in enum TargetQueryFilterField.

    Reason: Archived locations should not be queried for. Scheduled for removal after 2024/04/21.

  • Field allAwsComputeSettings was deprecated in Query type.

    Reason: This query is no longer supported. Scheduled for removal after 2024/04/29.

  • Field ownerOrgName was deprecated in GlobalSlaReply type.

    Reason: This field has been deprecated in favor of ownerOrg field. Scheduled for removal after 2024/05/11.
